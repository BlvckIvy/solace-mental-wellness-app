Solace is a student-focused mental wellness application designed to provide emotional support, self-reflection tools and access to helpful resources in a safe digital space.

## Problem
Many students struggle with their mental health but do not know where or how to seek help.

## Solution
Solace provides a private and supportive platform where students can:
- Reflect through guided journaling
- Access mental wellness resources
- Feel supported in a safe, non-judgmental space

## Features
- User authentication
- Guided journaling
- Mental wellness resources
- Safe-space focused UI design

## Tech Stack
- FlutterFlow
- Flutter (exported via FlutterFlow)
- Firebase (Authentication, Firestore)
